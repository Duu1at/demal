# Демал (Demal)

Мобильный маркетплейс туров по Кыргызстану. [cite_start]🇰🇬 Все туры в одном приложении. [cite: 8]

## 🎯 Проблема

[cite_start]Рынок локального туризма в Кыргызстане — это хаос.  Человек, желающий отдохнуть, сталкивается с тремя проблемами:

* [cite_start]**Информационный хаос:** Часы поиска подходящего тура в десятках разрозненных Instagram-аккаунтов и WhatsApp-групп. [cite: 5]
* [cite_start]**Отсутствие доверия:** Невозможно проверить надежность организатора, нет честных отзывов, а оплата уходит на личную карту незнакомцу. [cite: 6]
* [cite_start]**Потеря времени:** Бронирование превращается в долгую переписку, отправку скриншотов и ручное ожидание подтверждения. [cite: 7]

> [cite_start]"Я просто хотел отдохнуть, а не проводить расследование". 

## ✨ Решение

[cite_start]**"Demal"** — это мобильный маркетплейс, который собирает все предложения от проверенных организаторов туров в одном удобном приложении. [cite: 9] [cite_start]Мы как Booking.com или Aviasales, но для локального отдыха в Кыргызстане. [cite: 10]

Мы приносим порядок в хаос, предоставляя:
* [cite_start]**Выбор:** Все туры на любую дату в одной ленте. [cite: 12]
* [cite_start]**Удобство:** Простые фильтры и бронирование в три клика. [cite: 13]
* [cite_start]**Безопасность:** Проверенные организаторы, система рейтингов и безопасная онлайн-оплата. [cite: 14]

## 🚀 Ключевые функции (MVP)

### Для Клиентов
* [cite_start]Живая лента туров с фильтрами (дата, тип, цена). [cite: 19]
* [cite_start]Детальные карточки туров с программой, фото и отзывами. [cite: 20]
* [cite_start]Профили организаторов с системой рейтинга. [cite: 21]
* [cite_start]Безопасное онлайн-бронирование и оплата внутри приложения. [cite: 22]

* [cite_start]Регистрация и верификация профиля. [cite: 65, 67]
* [cite_start]Простое создание и управление турами. [cite: 70, 71]

## 🛠️ Технологический стек

* **Frontend:** Flutter
* **Архитектура:** Monorepo под управлением Melos

## 🏁 Начало работы

Инструкции для настройки и запуска проекта локально.

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <URL вашего репозитория>
    cd <название папки>
    ```

2.  **Убедитесь, что у вас установлен Flutter SDK.**

4.  **Установите Melos (если еще не установлен):**
    ```bash
    dart pub global activate melos
    ```

5.  **Установите зависимости и свяжите пакеты:**
    ```bash
    melos bootstrap
    ```

## 🏗️ Структура проекта

Этот проект использует архитектуру монорепозитория для разделения кода на логические модули.

* `/app`: Основное Flutter-приложение для клиентов, содержащее весь UI-код.
* `/packages`: Переиспользуемые Dart/Flutter пакеты (например, `api_client`, `data_models`, `shared_ui` и т.д.).
* `melos.yaml`: Файл конфигурации Melos для управления монорепозиторием.
