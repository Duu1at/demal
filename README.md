# Демал (Demal)

Мобильный маркетплейс туров по Кыргызстану. 🇰🇬 Все туры в одном приложении.

## 🎯 Проблема

Рынок локального туризма в Кыргызстане — это хаос.  Человек, желающий отдохнуть, сталкивается с тремя проблемами:

* **Информационный хаос:** Часы поиска подходящего тура в десятках разрозненных Instagram-аккаунтов и WhatsApp-групп.
* **Отсутствие доверия:** Невозможно проверить надежность организатора, нет честных отзывов, а оплата уходит на личную карту незнакомцу. 
* **Потеря времени:** Бронирование превращается в долгую переписку, отправку скриншотов и ручное ожидание подтверждения.

> "Я просто хотел отдохнуть, а не проводить расследование". 

## ✨ Решение

**"Demal"** — это мобильный маркетплейс, который собирает все предложения от проверенных организаторов туров в одном удобном приложении. [cite: 9] Мы как Booking.com или Aviasales, но для локального отдыха в Кыргызстане.

Мы приносим порядок в хаос, предоставляя:
* **Выбор:** Все туры на любую дату в одной ленте.
* **Удобство:** Простые фильтры и бронирование в три клика.
* **Безопасность:** Проверенные организаторы, система рейтингов и безопасная онлайн-оплата.

## 🚀 Ключевые функции (MVP)

### Для Клиентов
* Живая лента туров с фильтрами (дата, тип, цена).
* Детальные карточки туров с программой, фото и отзывами.
* Профили организаторов с системой рейтинга.
* Безопасное онлайн-бронирование и оплата внутри приложения.
* Личный кабинет с электронными "билетами" и историей поездок.

### Для Партнеров
* Регистрация и верификация профиля.
* Простое создание и управление турами.
* Просмотр списка участников в реальном времени.
* Внутренний кошелек и вывод заработанных средств.

## 🛠️ Технологический стек

* **Frontend:** Flutter
* **Архитектура:** Monorepo под управлением Melos

## 🏁 Начало работы

Инструкции для настройки и запуска проекта локально.

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <URL вашего репозитория>
    cd <название папки>
    ```

2.  **Убедитесь, что у вас установлен Flutter SDK.**

4.  **Установите Melos (если еще не установлен):**
    ```bash
    dart pub global activate melos
    ```

5.  **Установите зависимости и свяжите пакеты:**
    ```bash
    melos bootstrap
    ```

## 🏗️ Структура проекта

Этот проект использует архитектуру монорепозитория для разделения кода на логические модули.

* `/app`: Основное Flutter-приложение для клиентов, содержащее весь UI-код.
* `/packages`: Переиспользуемые Dart/Flutter пакеты (например, `api_client`, `data_models`, `shared_ui` и т.д.).
* `melos.yaml`: Файл конфигурации Melos для управления монорепозиторием.
